
かんたんなルール。

masterがメガビに持っていくブランチです。

linux使いますので、基本UTF-8の改行はLFのみで当面運用します。

フォルダは作り放題ですが、gameはルートフォルダのgamesに入れてください。
python-gamesのようにもともとゴチャゴチャに格納されているものはそのままで運用します。

ルートフォルダにゲーム以外のフォルダを作ってもかまいません（ツール等）

githubのコラボ登録済みメアドを持っている人は、gitpushする権利を有していると考えてください。

コミット失敗したら、git revert を使って「元に戻すためのコミット」をpushしてください。
履歴が荒れるとか、そういうことは気にしなくて大丈夫です。
